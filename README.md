# Tank Arena (Arcade)

Единая игра на `arcade` с режимами:

- локальный PvP (2 игрока на одной клавиатуре);
- сетевой режим через лобби (`host/client`, TCP);
- редактор арен с сохранением в `levels/*.json`;
- выбор пользовательских арен;
- туман войны (видимость только рядом с танком);
- быстрый запуск случайной карты.

## Быстрый старт

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## Управление

- Игрок 1: `WASD`, выстрел `SPACE`
- Игрок 2 (локально): стрелки, выстрел `Right Ctrl`
- В редакторе: ЛКМ ставит блок, ПКМ очищает, `1/2/3/4/0` выбор кисти, `S` сохранить, `G` сгенерировать случайную карту

## Ассеты

При первом запуске игра пытается загрузить PNG-ассеты в `assets/images/` из:

- `https://github.com/lolovlad/clientTangGame/tree/master/files/images`

Если загрузка недоступна, игра использует fallback-отрисовку цветами.

## Сеть / лобби

Сетевой режим использует локальный TCP-порт `57991`.

Для теста на одном ПК:
1. Запустить окно №1 и выбрать `Играть по сети -> Создать лобби`
2. Запустить окно №2 и выбрать `Играть по сети -> Подключиться к лобби`
